Project Overview

The E-Pharmacy Web Application is a full-stack online medicine management and shopping platform developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application enables users to browse medicines by category, search products, filter medicines based on price and category, manage shopping carts, and place orders. It also provides an administrative dashboard for managing categories, products, and users.

This project demonstrates the implementation of a complete e-commerce workflow specifically designed for pharmacy products, including product management, inventory control, cart functionality, and order processing.
